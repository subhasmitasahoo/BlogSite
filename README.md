# BlogSite
A website where one can create, Edit, Delete and View Blogs

This is a sample app made to demonstrate RESTful APIS and HTTP methods.

## Technologies used:
1. Front-end : 
   -HTML
   -CSS
   -Semantic-UI
2. Back-end : 
   -NodeJS
   -MongoDB
3. Packages used: 
   -body-parser
   -ejs, express
   -express-sanitizer
   -method-override
   -mongoose
